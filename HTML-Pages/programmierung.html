JavaScript

Kategorien von JavaScript-Tags
Grundlegende JavaScript-Tags
<script> - Definiert ein Skript-Element zum Einfügen von JavaScript-Code. <noscript> - Gibt an, dass der Inhalt nur angezeigt wird, wenn JavaScript deaktiviert ist. <link> - Stellt eine Verbindung zu externen Ressourcen, wie z. B. Stylesheets oder Skripten, her. <style> - Definiert eingebettete CSS-Styles im HTML-Dokument. <meta> - Definiert Metadaten wie Charset und Viewport-Informationen. <head> - Enthält Meta-Informationen, Links und Skripte, jedoch keinen sichtbaren Inhalt.
Ereignisse und Interaktivität
addEventListener() - Fügt ein Event-Listener zu einem Element hinzu. alert() - Zeigt ein modales Dialogfeld mit einer Nachricht an. confirm() - Zeigt ein modales Dialogfeld mit einer Bestätigungsmeldung an. prompt() - Zeigt ein Dialogfeld an, in dem der Benutzer Eingaben machen kann. setTimeout() - Führt eine Funktion nach einer Verzögerung aus. setInterval() - Führt eine Funktion in regelmäßigen Abständen aus.
clearTimeout() - Stoppt die Ausführung einer durch setTimeout() gesetzten Funktion. clearInterval() - Stoppt die wiederholte Ausführung einer durch setInterval() gesetzten Funktion. document.getElementById() - Gibt das Element mit der angegebenen ID zurück. document.querySelector() - Gibt das erste Element zurück, das dem angegebenen CSS-Selektor entspricht. document.querySelectorAll() - Gibt alle Elemente zurück, die dem angegebenen CSS-Selektor entsprechen. getAttribute() - Gibt den Wert eines Attributs eines HTML-Elements zurück.
Objekte und Arrays
Array() - Erzeugt ein neues Array. Object() - Erzeugt ein neues Objekt. JSON.parse() - Wandelt einen JSON-String in ein JavaScript-Objekt um. JSON.stringify() - Wandelt ein JavaScript-Objekt in einen JSON-String um. Math.random() - Gibt eine Zufallszahl im Bereich von 0 bis 1 zurück. Math.floor() - Rundet eine Zahl ab.
Math.ceil() - Rundet eine Zahl auf. Math.max() - Gibt den größten Wert unter den angegebenen Werten zurück. Math.min() - Gibt den kleinsten Wert unter den angegebenen Werten zurück. Math.sqrt() - Gibt die Quadratwurzel einer Zahl zurück. Array.push() - Fügt ein oder mehrere Elemente ans Ende eines Arrays an. Array.pop() - Entfernt das letzte Element aus einem Array.
DOM-Manipulation
document.createElement() - Erstellt ein neues HTML-Element. document.createTextNode() - Erstellt einen neuen Textknoten. document.appendChild() - Fügt ein Element als Kind eines anderen Elements hinzu. document.removeChild() - Entfernt ein Kind-Element von einem übergeordneten Element. element.classList.add() - Fügt einer Liste von Klassen eine neue Klasse hinzu. element.classList.remove() - Entfernt eine Klasse aus der Liste der Klassen eines Elements.
element.style.property - Ändert den Stil eines Elements direkt. element.setAttribute() - Setzt den Wert eines Attributs eines HTML-Elements. element.getAttribute() - Gibt den Wert eines Attributs eines HTML-Elements zurück. element.innerHTML - Holt oder setzt den HTML-Inhalt eines Elements. element.textContent - Holt oder setzt den Textinhalt eines Elements. element.style - Ermöglicht das direkte Setzen von CSS-Stilen für ein Element.
Fehlerbehandlung
try...catch - Behandelt Fehler in einem Codeblock. throw - Wirft eine benutzerdefinierte Ausnahme. finally - Wird nach dem Ausführen eines try...catch-Blocks ausgeführt, unabhängig davon, ob ein Fehler aufgetreten ist. console.log() - Gibt Informationen in der Konsole aus. console.error() - Gibt Fehlernachrichten in der Konsole aus. console.warn() - Gibt Warnmeldungen in der Konsole aus.
Asynchrone Programmierung
Promise() - Erzeugt ein Promise-Objekt, das einen zukünftigen Wert darstellt. async - Kennzeichnet eine Funktion als asynchron. await - Wartet auf die Auflösung eines Promises in einer asynchronen Funktion. fetch() - Macht eine HTTP-Anfrage und gibt ein Promise zurück. setTimeout() - Verzögert die Ausführung einer Funktion um eine angegebene Zeit. setInterval() - Führt eine Funktion wiederholt in festgelegten Intervallen aus.
Array-Methoden
Array.slice() - Gibt einen Teil des Arrays zurück, ohne das Original zu verändern. Array.splice() - Ändert den Inhalt eines Arrays durch Hinzufügen, Entfernen oder Ersetzen von Elementen. Array.forEach() - Führt eine Funktion für jedes Element im Array aus. Array.map() - Erstellt ein neues Array, das durch Anwenden einer Funktion auf jedes Element des Originalarrays entsteht. Array.filter() - Erstellt ein neues Array mit allen Elementen, die eine Bedingung erfüllen. Array.reduce() - Reduziert das Array auf einen einzigen Wert, indem eine Funktion auf jedes Element angewendet wird.
Array.reduceRight() - Reduziert das Array von rechts nach links auf einen einzigen Wert. Array.find() - Gibt das erste Element im Array zurück, das eine Bedingung erfüllt. Array.findIndex() - Gibt den Index des ersten Elements im Array zurück, das eine Bedingung erfüllt. Array.indexOf() - Gibt den Index des ersten Vorkommens eines bestimmten Werts im Array zurück. Array.includes() - Überprüft, ob ein Array ein bestimmtes Element enthält. Array.concat() - Kombiniert mehrere Arrays oder Werte in ein neues Array.
String-Methoden

String.includes() - Überprüft, ob ein String ein bestimmtes Teilzeichen enthält. String.split() - Teilt einen String anhand eines Trennzeichens und gibt ein Array zurück. String.slice() - Gibt einen Teil des Strings zurück, ohne das Original zu verändern. String.replace() - Ersetzt ein Vorkommen eines Musters durch einen neuen Wert.
String.toLowerCase() - Wandelt den String in Kleinbuchstaben um. String.toUpperCase() - Wandelt den String in Großbuchstaben um. String.trim() - Entfernt Leerzeichen am Anfang und Ende eines Strings. String.charCodeAt() - Gibt den Unicode-Wert des Zeichens an einem bestimmten Index zurück. String.substr() - Gibt einen bestimmten Teil eines Strings zurück, beginnend an einem bestimmten Index. String.repeat() - Wiederholt den String eine angegebene Anzahl von Malen.

Math.random() - Gibt eine zufällige Zahl im Bereich von 0 bis 1 zurück. Math.min() - Gibt den kleinsten Wert unter den angegebenen Werten zurück. Math.max() - Gibt den größten Wert unter den angegebenen Werten zurück.

Math.sin() - Gibt den Sinus eines Winkels (in Bogenmaß) zurück. Math.cos() - Gibt den Kosinus eines Winkels (in Bogenmaß) zurück. Math.tan() - Gibt den Tangens eines Winkels (in Bogenmaß) zurück.
Date und Zeit
Date() - Erstellt ein neues Date-Objekt mit dem aktuellen Datum und der aktuellen Uhrzeit. Date.now() - Gibt die Anzahl der Millisekunden seit dem 1. Januar 1970 zurück. Date.getFullYear() - Gibt das Jahr eines Date-Objekts zurück. Date.getMonth() - Gibt den Monat (0-11) eines Date-Objekts zurück. Date.getDate() - Gibt den Tag des Monats (1-31) eines Date-Objekts zurück. Date.getHours() - Gibt die Stunde (0-23) eines Date-Objekts zurück.
Date.getMinutes() - Gibt die Minuten (0-59) eines Date-Objekts zurück. Date.getSeconds() - Gibt die Sekunden (0-59) eines Date-Objekts zurück. Date.getMilliseconds() - Gibt die Millisekunden (0-999) eines Date-Objekts zurück. Date.setFullYear() - Setzt das Jahr eines Date-Objekts. Date.setMonth() - Setzt den Monat eines Date-Objekts. Date.setDate() - Setzt den Tag des Monats eines Date-Objekts.
